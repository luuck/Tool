# tool
插件和小工具的集合

## 背景介绍（introduce）
因为工作的原因，需要兼容ie低版本的游览器，所以收集、整理、开发了一些可复用的插件和小工具，兼容ie6+以及各大主流游览器（chrome,ff,360.）。这些插件的实现方式当然是不唯一的，不过都拥有更好的##兼容性##！
